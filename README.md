# # AfyaConnect

### A Digital Health Access Platform for Kenyans

🌐 **Live Demo:** [https://my-afya-link.vercel.app/](https://my-afya-link.vercel.app/)
📱 **APK Version:** Available (Android)
👥 **Team:** Health Action Team
🏥 **Sector:** Health

---

## 📖 About the Project

**AfyaConnect** is a mobile-first digital health platform designed to improve access to reliable health information and medical guidance for underserved communities in Kenya.

The platform provides:

* ✅ Verified health information
* 🩺 Symptom guidance
* 📍 Nearby clinic location services
* 📲 SMS-based health alerts and appointment reminders

AfyaConnect addresses healthcare access gaps in rural and low-income communities by combining web and SMS technologies to deliver trusted, actionable health support.

---

## 📱 Mobile Application (APK Deployment)

To enhance accessibility and usability on smartphones, the AfyaConnect web application was converted into a native Android application (APK) using Median.

This approach enables:

* 📲 Installation on Android devices
* 🚀 Faster access without opening a browser
* 📱 Improved mobile user experience
* 🔄 Seamless interaction with the web platform

The APK version maintains all core functionalities of the web application while providing a more app-like experience.

---

## 🚨 Problem Statement

Many Kenyans—especially those in rural and low-income areas—struggle to access reliable health information and timely medical guidance. This leads to:

* Delayed treatment
* Preventable illnesses
* Reliance on unverified sources
* Poor appointment adherence

Limited access to trustworthy digital health systems significantly impacts health outcomes across communities.

---

## 💡 Solution Overview

AfyaConnect delivers:

* A mobile-first web platform
* A symptom guidance interface
* A verified medical information repository
* A clinic discovery and location system
* An SMS-based appointment and health reminder service

The system is accessible to both smartphone users and individuals using basic mobile phones via SMS.
